# TUNA Blogroll

世界一流兼容并包 TUNA 协会收集的周围同学们的 Blog。

## FAQ

> 萌新也可以加 blog 列表么？

能。

> 有些 blog 太久没更新或失效了，怎么办？

请提 pull request 删掉

## 添加方式

在**最下面一行**添加（相当于按时间顺序，为以后的增量提醒做准备）。

TUNA 的同学们有权限可以直接编辑此文件；其它同学们烦请发 pull request。

推荐在 commit log 或者 pull request 里面简单介绍一下自己，比如常用的 ID 等。
